Customer Feedback Sentiment Analysis Dashboard Overview

This project combines Natural Language Processing (NLP) and Power BI visualisation to analyse customer feedback data. Using Python, customer comments were processed through sentiment analysis and keyword extraction techniques before being transformed into an interactive Power BI dashboard for business insight generation.

The workflow automates the extraction of customer sentiment and recurring themes from survey responses, enabling more efficient analysis of customer experience data.

Objectives
- Analyse customer feedback sentiment
- Extract recurring customer phrases and keywords
- Automate text processing using NLP techniques
- Transform processed feedback into interactive business dashboards
- Support customer insight and decision-making through data visualisation

Technologies Used
- Python
- pandas
- VADER Sentiment Analysis
- KeyBERT
- Power BI
- NLP (Natural Language Processing)
- Excel

Workflow
The project workflow includes:
1. Loading customer feedback data from Excel
2. Cleaning and preprocessing text responses
3. Performing sentiment analysis using VADER
4. Extracting keywords and phrases using KeyBERT
5. Exporting processed results to Excel
6. Building an interactive Power BI dashboard using the transformed dataset

Dataset Features
The analysis uses customer feedback fields including:
- Date
- Demographic information
- Recommendations
- Overall customer feedback

Key Findings
- Customer feedback was successfully classified into:
  - Positive
  - Neutral
  - Negative sentiment categories
- Phrase extraction identified recurring themes and commonly mentioned customer concerns and experiences.
- The Power BI dashboard enabled interactive exploration of:
  - Sentiment trends
  - Frequently occurring phrases
  - Customer feedback distribution
  - Demographic insights
- NLP automation significantly reduced the manual effort required to analyse large volumes of qualitative feedback data.
- Combining Python-based NLP processing with Power BI visualisation improved the accessibility and interpretability of customer feedback insights.
